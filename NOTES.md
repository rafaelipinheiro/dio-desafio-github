# GIT/GITHUB

## Verificar a instalação
`git --version`

## Sua identidade
`git config --global user.name “Seu nome”`
`git config --global user.email seu@email`

## Visualizar configurações
`cat .gitconfig`

## Criar um repositório
`git init`

## Ajuda
`git help`
